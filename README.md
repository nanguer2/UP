## ⛓️ Guía de Correlativas y "Materias Llave"
Basado en el **Plan de Estudios Reorganizado 2026**, estas son las materias críticas que habilitan el avance en las distintas ramas de la ingeniería:

### 🧮 Rama de Matemáticas y Señales
* [cite_start]**Análisis Matemático IIIb (025054):** Habilita directamente *Señales y Sistemas* y *Matemática Avanzada*[cite: 7].
* [cite_start]**Matemática Avanzada (026435):** Requisito indispensable para cursar *Electromagnetismo*[cite: 15, 22].
* [cite_start]**Probabilidad y Estadística (025915):** Habilita la materia de *Procesos Estocásticos*[cite: 11].
* [cite_start]**Álgebra Lineal (023209):** Es la base necesaria para *Introducción a la Robótica*[cite: 9, 59].

### ⚡ Rama de Física y Electrónica
* [cite_start]**Física I (026095):** Habilita cursar tanto *Física IIa* como *Física IIb*[cite: 6].
* [cite_start]**Física IIb (026204):** Es la llave para *Dispositivos Electrónicos*, *Teoría de Circuitos* y *Laboratorio de Electrónica*[cite: 13, 19, 28].
* [cite_start]**Electromagnetismo (026446):** Requisito para *Conducción e Irradiación de Ondas*[cite: 35].

### 💻 Rama de Sistemas y Computación
* [cite_start]**Arquitectura de Computadores (025612):** Habilita las electivas de *Sistemas Operativos* e *Introducción a los Sistemas Embebidos*[cite: 16, 23, 32].
* [cite_start]**Sistemas Digitales II (026178):** Necesaria para ingresar al *Laboratorio de Microprocesadores*[cite: 10, 21].
